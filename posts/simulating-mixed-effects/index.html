Simulating Mixed Effects

[Updated 2019-01-03]

library(tidyverse)
library(lmerTest)

I’m going to walk through one example of simulating a dataset with random effects. I’ll generate data for a study where subjects are in one of two conditions (A and B) and respond to stimuli shown in each of two versions (congruent and incongruent). The DV is reaction time.

Subjects

First, we need to generate a sample of subjects. Each subject will have slightly faster or slower reaction times on average; this is their random intercept (sub_i). We’ll model it from a normal distribution with a mean of 0 and SD of 100ms (you can choose a reasonable SD for your design from pilot data by calculating each subject’s mean DV and using the SD).

We also add between-subject variables here. Each subject is in only one condition, so assign half A and half B. Set the number of subjects as sub_n at the beginning so you can change this in the future with only one edit.

sub_n  <- 200 # number of subjects in this simulation
sub_sd <- 100 # SD for the subjects' random intercept

sub <- tibble(
  sub_id = 1:sub_n,
  sub_i  = rnorm(sub_n, 0, sub_sd) %>% round(), # random intercept
  sub_cond = rep(c("A","B"), each = sub_n/2)
)
If you already have pilot data, you can estimate an SD for the subjects’ random intercepts using the following code. 
pilot_data <- read_csv("pilot_data.csv")
pilot_mod <- lmer(dv ~ 1 + (1 | sub_id) + (1 | stim_id), data = pilot_data)

sub_sd <- VarCorr(pilot_mod) %>% 
  as.data.frame() %>%
  filter(grp == "sub_id") %>% 
  pull(sdcor)

I like to check my simulations at every step with a graph. We expect subjects in conditions A and B to have approximately equal intercepts.

ggplot(sub, aes(sub_i, color = sub_cond)) +
  geom_density()

Stimuli

Now, we generate a sample of stimuli. Each stimulus will have slightly faster or slower reaction times on average; this is their random intercept (stim_i). We’ll model it from a normal distribution with a mean of 0 and SD of 50ms. As above, you can calculate the mean Dv for each stimulus from pilot data and use the SD. Stimulus version (congruent vs incongruent) is a within-stimulus variable, so we don’t need to add it here.

stim_n  <- 50 # number of stimuli in this simulation
stim_sd <- 50 # SD for the stimuli's random intercept

stim <- tibble(
  stim_id = 1:stim_n,
  stim_i = rnorm(stim_n, 0, stim_sd) %>% round()
)
If you want to simulate data using the exact stimuli you already have pilot data for, you can calculate their random intercepts using the following code and use that instead of generating a random sample with the same SD. 
pilot_data <- read_csv("pilot_data.csv")
pilot_mod <- lmer(dv ~ 1 + (1 | sub_id) + (1 | stim_id), data = pilot_data)

stim <- ranef(pilot_mod)$stim_id %>%
  as_tibble(rownames = 'stim_id') %>%
  rename(stim_i = `(Intercept)`)

Plot the random intercepts to double-check they look like you expect.

ggplot(stim, aes(stim_i)) +
  geom_density()

Trials

Now we put the subjects and stimuli together. In this study, all subjects respond to all stimuli in both upright and inverted versions, but subjects are in only one condition. The function expand.grid gives you a data frame with all possible combinations of the arguments. Add the data specific to each subject and stimulus by left joining the sub and stim data frames.

trials <- expand.grid(
  sub_id = sub$sub_id, # get subject IDs from the sub data table
  stim_id = stim$stim_id, # get stimulus IDs from the stim data table
  stim_version = c("congruent", "incongruent") # all subjects see both congruent and incongruent versions of all stimuli
) %>%
  left_join(sub, by = "sub_id") %>% # includes the intercept and conditin for each subject
  left_join(stim, by = "stim_id")   # includes the intercept for each stimulus

Calculate DV

Now we can calculate the DV by adding together an overall intercept (mean RT for all trials), the subject-specific intercept, the stimulus-specific intercept, the effect of subject condition, the effect of stimulus version, and an error term (we’re not including an interaction between condition and version here).

We set these effects in raw units (ms) and effect-code the subject condition and stimulus version. It’s usually easiest to interpret if you recode the level that you predict will be larger as +0.5 and the level you predict will be smaller as -0.5. So when we set the effect of subject condition (sub_cond_eff) to 50, that means the average difference between condition A and condition B is 50ms. Condition A is effect-coded as -0.5 and condition B is effect-coded as +0.5, which means that trials in condition A have -0.5 * 50ms (i.e., -25ms) added to their reaction time, while trials in condition B have +0.5 * 50ms (i.e., +25ms) added to their reaction time.

# set variables to use in calculations below
grand_i          <- 400 # overall mean DV
sub_cond_eff     <- 50  # mean difference between conditions B and A
stim_version_eff <- 50  # mean difference between versions incongruent and congruent
error_sd         <- 25  # residual (error) SD
We set the error SD fairly low here so that it’s easier to see how the parameters we set map onto the analysis output. You can calculate a realistic error SD from pilot data with the code below. 
pilot_data <- read_csv("pilot_data.csv")
pilot_mod <- lmer(dv ~ 1 + (1 | sub_id) + (1 | stim_id), data = pilot_data)

error_sd <- VarCorr(pilot_mod) %>% 
  as.data.frame() %>%
  filter(grp == "Residual") %>% 
  pull(sdcor)
dat <- trials %>%
  mutate(
    # effect-code subject condition and stimulus version
    sub_cond.e = recode(sub_cond, "A" = -0.5, "B" = +0.5),
    stim_version.e = recode(stim_version, "congruent" = -0.5, "incongruent" = +0.5),
    # calculate error term (normally distributed residual with SD set above)
    err = rnorm(nrow(.), 0, error_sd) %>% round(),
    # calculate DV from intercepts, effects, and error
    dv = grand_i + sub_i + stim_i + 
         (sub_cond.e * sub_cond_eff) + 
         (stim_version.e * stim_version_eff) + 
         err
  )

Let’s take a concrete example.

sub_id stim_id stim_version sub_i sub_cond stim_i sub_cond.e stim_version.e err dv
1 1 congruent 100 A -36 -0.5 -0.5 -2 412

The DV is equal to the grand intercept (400) plus the subject’s intercept (100) plus the stimulus’ intercept (-36) plus the effect code for the subject’s condition (A = -0.5) times the effect size for subject condition (50) plus the effect code for stimulus version (congruent = -0.5) times the effect size for stimulus version (50) plus some random error (-2).

400 + 100 + -36 + (-0.5 * 50) + (-0.5 * 50) + -2 = 412

As always, graph to make sure you’ve simulated the general pattern you expected.

ggplot(dat, aes(sub_cond, dv, color = stim_version)) +
  geom_hline(yintercept = grand_i) +
  geom_violin(alpha = 0.5) +
  geom_boxplot(width = 0.2, position = position_dodge(width = 0.9))

In this simulated dataset, the grand intercept is 398.4, the mean difference between subject conditions A and B is 57.4, and the mean difference between stimulus versions is 49.7.

More Complicated Patterns

If you want to simulate an interaction, it can be tricky to figure out what to set the main effects and interaction effect to. It can be easier to think about the simple main effects for each cell. Create four new variables and set them to the deviations from the overall mean you’d expect for each condition (so they should add up to 0). Here, we’re simulating a small effect of version in condition A (50ms difference) and double that effect of version in condition B (100ms difference).

# set variables to use in calculations below
grand_i          <- 400
cond_a_version_c <- -25
cond_a_version_i <- +25
cond_b_version_c <- -50
cond_b_version_i <- +50
error_sd         <-  25

Use the code below to transform the simple main effects above into main effects and interactions for use in the equations below.

# calculate main effects and interactions from simple effects above

# mean difference between conditions B and A
sub_cond_eff     <- (cond_b_version_c + cond_b_version_i)/2 -
                    (cond_a_version_c + cond_a_version_i)/2
# mean difference between versions incongruent and congruent
stim_version_eff <- (cond_a_version_i + cond_b_version_i)/2 - 
                    (cond_a_version_c + cond_b_version_c)/2  
# interaction between version and condition
cond_version_ixn <- (cond_b_version_i - cond_b_version_c) -
                    (cond_a_version_i - cond_a_version_c)

Then generate the DV the same way we did above, but also add the interaction effect multiplied by the effect-coded subject condition and stimulus version.

dat <- trials %>%
  mutate(
    # effect-code subject condition and stimulus version
    sub_cond.e = recode(sub_cond, "A" = -0.5, "B" = +0.5),
    stim_version.e = recode(stim_version, "congruent" = -0.5, "incongruent" = +0.5),
    # calculate error term (normally distributed residual with SD set above)
    err = rnorm(nrow(.), 0, error_sd) %>% round(),
    # calculate DV from intercepts, effects, and error
    dv = grand_i + sub_i + stim_i + 
         (sub_cond.e * sub_cond_eff) + 
         (stim_version.e * stim_version_eff) + 
         (sub_cond.e * stim_version.e * cond_version_ixn) +
         err
  )

First, always check that your groups make sense. The number of obs should be the total nuber of trials analysed. sub_id should be what we set sub_n to above (200) and stim_id should be what we set stim_n to above (50).

Next, look at the random effects.

  • The SD for sub_id should be near the sub_sd of 100.
  • he SD for stim_id should be near the stim_sd off 50.
  • The residual SD should be near the error_sd of 25.

Finally, look at the fixed effects.

  • The estimate for the Intercept should be near the grand_i of 400.
  • The main effect of sub_cond.e should be near what we calculated for sub_cond_eff (0).
  • The main effect of stim_version.e should be near what we calculated for stim_version_eff (75)
  • The interaction between sub_cond.e:stim_version.e should be near what we calculated for cond_version_ixn (50).

Function

You can put the above in a function so you can run it more easily. I removed the plot and set the argument defaults to the same as the examples above, but you can set them to other patterns.

sim_lmer <- function( sub_n = 200,
                      sub_sd = 100,
                      stim_n = 50,
                      stim_sd = 50,
                      grand_i = 400,
                      cond_a_version_c = -25,
                      cond_a_version_i = +25,
                      cond_b_version_c = -50,
                      cond_b_version_i = +50,
                      error_sd = 25) {
  sub <- tibble(
    sub_id = 1:sub_n,
    sub_i  = rnorm(sub_n, 0, sub_sd),
    sub_cond = rep(c("A","B"), each = sub_n/2)
  )
  
  stim <- tibble(
    stim_id = 1:stim_n,
    stim_i = rnorm(stim_n, 0, stim_sd)
  )
  
  # calculate main effects and interactions from simple effects above
  
  # mean difference between conditions B and A
  sub_cond_eff     <- (cond_b_version_c + cond_b_version_i)/2 -
                      (cond_a_version_c + cond_a_version_i)/2
  # mean difference between versions incongruent and congruent
  stim_version_eff <- (cond_a_version_i + cond_b_version_i)/2 - 
                      (cond_a_version_c + cond_b_version_c)/2  
  # interaction between version and condition
  cond_version_ixn <- (cond_b_version_i - cond_b_version_c) -
                      (cond_a_version_i - cond_a_version_c) 
  
  dat <- expand.grid(
    sub_id = sub$sub_id,
    stim_id = stim$stim_id,
    stim_version = c("congruent", "incongruent")
  ) %>%
    left_join(sub, by = "sub_id") %>%
    left_join(stim, by = "stim_id") %>%
    mutate(
      # effect-code subject condition and stimulus version
      sub_cond.e = recode(sub_cond, "A" = -0.5, "B" = +0.5),
      stim_version.e = recode(stim_version, "congruent" = -0.5, "incongruent" = +0.5),
      # calculate error term (normally distributed residual with SD set above)
      err = rnorm(nrow(.), 0, error_sd) %>% round(),
      # calculate DV from intercepts, effects, and error
      dv = grand_i + sub_i + stim_i + 
           (sub_cond.e * sub_cond_eff) + 
           (stim_version.e * stim_version_eff) + 
           (sub_cond.e * stim_version.e * cond_version_ixn) +
           err
    )
  
  mod <- lmer(dv ~ sub_cond.e * stim_version.e +
                (1 | sub_id) + 
                (1 | stim_id),
              data = dat)
  
  mod.sum <- summary(mod)
  
  return(mod.sum)
}
